# Changelog

All notable changes to FlaggerLink's legal documents are documented in this file.

## [v1.1] - 2026-02-19

### Terms of Service
- Added license grant and credential sharing provisions (§3)
- Added prohibited uses and competitive use restrictions (§4)
- Expanded billing section with plan-based tiers and no-refund policy (§5)
- Added SMS and email compliance sections with TCPA, CAN-SPAM, and CTIA provisions (§6)
- Added customer data ownership and data security with 72-hour breach notification (§9–§10)
- Added confidentiality obligations (§11)
- Added warranty disclaimers (§12)
- Added limitation of liability with exceptions for fraud, gross negligence, and willful misconduct (§13)
- Added indemnification (§14)
- Added force majeure (§15)
- Expanded termination section with liquidated damages formula (§16)
- Added AAA binding arbitration with class action waiver and injunctive relief carve-outs (§17)
- Added general provisions: severability, assignment, notices, entire agreement (§18)

### Privacy Policy
- Added B2B scope and applicability section (§1)
- Clarified data ownership and data processor/controller model
- Added employee self-service features clarification
- Added Cloudflare Insights to analytics services
- Added Do Not Track signal disclosure
- Expanded data sharing section with explicit "no sale" language
- Added TCPA-compliant SMS messaging terms
- Added data retention schedule by category
- Added US-only data transfer disclosure
- Added data deletion and export provisions (payment required)
- Added CCPA/CPRA section with categories table, sensitive PI, and rights
- Restructured privacy rights handling through Customer organizations
